About Media Queries

Media queries let the webpage change depending on the condition. The @media lets certain CSS properties defined under it show when certain criteria are met, such as the minimum webapge width. A min-width query makes the CSS under the query visible if the window is bigger than the size specified. A max-width query makes the CSS under the query visible if the window is smaller than the size specified. We often start designing on a smaller screen then design up to larger screens (called mobile first design) because it makes for better experience for mobile users and most users are mobile anyways.

Relative Units

We often use relative units (such as rem, em, vw, vh) in web design because they change based on the size of the based on another length. For example, vw and vh change based on the viewport size, which is helpful when designing for different sized windows. Em is based on parent font sizes so it can be a percentage of how big its parent font is.

Meta Viewport Tag

A meta viewport tag is used control how a page scales based on how big the viewport is, or how much of a webpage a person can see. We write it at the top of the page to determine how big the user's viewport is and we can use css to adjust the page accordingly. When the meta viewport tag is there and the page is seen in the inspector, you can see the page with the text. Without it, only the top left of the large webpage can be seen.
