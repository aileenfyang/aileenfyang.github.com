Why we use the Normalize CSS Script

We use normalize to make sure that all elements are consistent when opening with different browsers. That way,no matter if the person is using Chrome or Firefox, they can see the website the way that the designer intended for it to be seen.

What is Semantic Markup

Semantic markup keeps the elements of a webpage organized and creates a sort of hierarchy. It makes reading and using a website easy and seamless. It keeps everything in in the website structured so there is less confusion for both the person writing it and the person using it.
