Structure, Phrasing, and Display

Structural (block) elements display the full length of the horizontal based on its parent and they begin on new lines. Phrasing (inline) elements can start in the middle of a line and it doesn't start a new line. It only goes as far horizontally as its tags define. The different settings for display include block, inline, grid, none, and more. They all control layout in a different way. They dispay things in different areas of the screen. For example grid can put material in any box that is specified in the grid.

Box Model

The CSS box model shows the layers that wrap each element. It goes (from inside to outside) content, padding, border, then margin. Content is where the content, like text or images appear. Padding is a transparent area around the content. Borders go around the padding, and margins are transparent areas around the border.The box-sizing property includes padding and border into the width and height of an element, making the element bigger than it actually is.

Background Images

An image using the img tag in html will load before a background image placed by the css. A background image is not part of the content. Images using the img tag in html are part of the content of the page.
