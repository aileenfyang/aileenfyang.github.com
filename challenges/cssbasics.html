Inheritance in CSS

Inheritance in CSS is when a value is not specified on an element. Some elements will inherit the properties of the parent element, while others will not. Some properties that are inherited include color, font-size, font-style, and font.

Specificity in CSS

When an element has multiple rules that are pointed at it, specificity deals with which rule is applied. If there are two rules of equal specificity, the one that comes last will be used. There is a hierarchy for specificity. For example, universal selectors are very low on the hierarchy for specificity while ID tags are very specific.

ID and Class Attributes

IDs are used for uniques elements, ones that are not going to be used again on the page. Classes are used for things that are specific, but used throughout the page. A class is designated in css as .classname{} while an ID is designated as #IDname{}.
